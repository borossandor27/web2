Using the Fetch API

A Fetch egy böngésző API szövegek, képek, strukturált adatok aszinkron betöltésére a HTML-oldalak frissítéséhez. Abban tér el a már megismert XMLHttpRequest-től, hogy a Promise objektumra épül, ami nagymértékben leegyszerűsíti a kódot, különösen, ha az async/await funkcióval együtt használjuk.
